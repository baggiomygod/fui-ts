http://example.com/text#fragment,#fragment就是片段识别符。

如果只是改变片段标识符，页面不会刷新。

1. origin.html:父页面设置iframe元素而的src属性

      let originUrl = 'http://localhost:8080/';
      let btnEl = document.getElementById('btn');
      btnEl.addEventListener('click', changeHash);
      // 父页面设置iframe元素而的src属性
      function changeHash(){
        let i = 0;
        i++;
        let src = originUrl + '#/' + i;
        let iframe = document.getElementById('my-iframe');
        iframe.src = src;
      }

2. header.vue:子页面监听hash变化拿到数据

       checkHashChange(){
         let hash = window.location.hash;
         this.title = hash;
       }
       mounted(){
         window.onhashchange = this.checkHashChange;
       }
